# medcodes

Medical code ontologies, hierarchy traversal, and cross-system mapping.

## Features

- `icd10cm` - ICD-10-CM code definitions (default)
- `serde` - Serialization support for all code types

## Usage

```rust
use medcodes::icd10::ICD10CM;
```
